Parity¶

Parity claims to be the world’s fastest and lightest Ethereum client. It is written in the Rust language, which offers improved reliability, performance, and code clarity. Parity is being developed by Parity Technologies (f.k.a. Ethcore), which was founded by several members of the Ethereum Foundation.

  • Website: https://parity.io/
  • GitHub: https://github.com/paritytech/parity
  • Gitter chat: https://gitter.im/paritytech/parity

Arch Linux packages are maintained by Afri Schoedon.

  • https://www.archlinux.org/packages/community/x86_64/parity/

Some people have reported success with Parity on Raspberry Pi 2.
